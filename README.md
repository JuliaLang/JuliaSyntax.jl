# JuliaSyntax

The new julia parser.  Active development has moved to [the main julia repository](https://github.com/JuliaLang/julia) in versions 1.14+ [(see julia PR 59870)](https://github.com/JuliaLang/julia/pull/59870).  New issues and PRs should be created there.

Backports to Julia version <=1.13 are still done in this repository:
- backports-julialang-1.12
- backports-julialang-1.13

Backports to `JuliaSyntax.jl`-the-package version <=1 are also done here:
- [backports-release-1](https://github.com/JuliaLang/JuliaSyntax.jl/tree/backports-release-1)
